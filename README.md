# 📖 프로젝트 소개
> "TemPo"는 집에서 운동을 하고싶은 운동인들을 위한 통합 웹 서비스입니다. <br><br>사용자를 위한 운동 영상 추천은 물론, 서로 소통하고 정보를 공유함으로써 운동에 대한 편의성과 접근성을 높일 수 있습니다. 템포와 함께, 더 건강하고 활기찬 삶의 시작을 만나보세요.
